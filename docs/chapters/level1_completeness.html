3 🌊 Level 1: Completeness of the Real Numbers

Lila:
“Acharya, we already have natural numbers, integers, rationals. Why do we need real numbers? Aren’t rationals enough?”

Acharya Bhaskara:
“That is a wise question, Lila. Let us test the rationals. Can you think of a number whose square is 2?”

Lila:
“Well… if I try,
- \(1.4^2 = 1.96\),
- \(1.41^2 = 1.9881\),
- \(1.414^2 = 1.999396\)…
We get closer, but never exactly 2.”

Acharya Bhaskara:
“So, do you think there is a rational number \(q\) with \(q^2 = 2\)?”

Lila:
“No. I’ve seen the proof that \(\sqrt{2}\) is irrational.”

Acharya Bhaskara:
“Correct. But notice this: even if \(\sqrt{2}\) is not rational, we can approximate it as closely as we like using rationals — \[1.4, 1.41, 1.414\ldots\] The rationals are dense: between any two rationals there is another. Yet something is still missing.”

3.1 🔹 Boundedness

Acharya Bhaskara:
“Let us capture this rigorously. Consider the set

\[ S = \{ x \in \mathbb{Q} : x^2 < 2 \}. \]

Is there a number that is always greater than every element of this set?”

Lila:
“Sure, 2 is bigger than all of them. So is 1.5, or 1.42.”

Acharya Bhaskara:
Exactly. We call such numbers upper bounds. Here is a precise definition.

Definition (Upper bound).
A number \(M\) is an upper bound of a set \(S \subseteq \mathbb{R}\) if \[ x \leq M \quad \text{for all } x \in S. \]

Similarly, we can define a lower bound.

Definition (Lower bound).
A number \(m\) is an upper bound of a set \(S \subseteq \mathbb{R}\) if \[ x \geq M \quad \text{for all } x \in S. \]

Lila:
“Acharya, you defined what it means for a number to be an upper bound. But what about the set itself? When do we call a set bounded?”

Acharya Bhaskara:
“A good question, Lila. Here is the precise definition.”

Definition (Bounded Set).
A set \(S \subseteq \mathbb{R}\) is said to be bounded above if it has an upper bound.
It is bounded below if it has a lower bound.
If \(S\) is both bounded above and below, then \(S\) is called a bounded set.
Formally, there exist real numbers \(m, M \in \mathbb{R}\) such that \[ m \le x \le M \quad \text{for all } x \in S. \]

3.2 🔹 Examples

Example 1.
The interval [0,1] is bounded above by 1, 2, \(\ldots\) and bounded below by 0, -1, \(\ldots\).
In fact, 0 and 1 are the tightest bounds, so the set is bounded.

Example 2.
The set \(\mathbb{N} = \{1,2,3,\ldots\}\) is bounded below (by \(1, 0, -5, \ldots\)),
but it is not bounded above — it grows without end.

Example 3.
The set \(\mathbb{Z}\) of all integers has no upper bound and no lower bound.
Thus, it is not bounded.

Lila:
“So a set is bounded if it fits within some finite interval \([m,M]\) on the number line?”

Acharya Bhaskara:
“Exactly! That’s the intuition — bounded sets live inside some finite cage on the real line.
Unbounded sets escape that cage, either upward, downward, or both.”

Acharya Bhaskara:
“But among upper bounds some are tighter than others. Which number feels like the ‘tightest’ upper bound for our set $ S = { x : x^2 < 2 }. $?”

Lila:
“It should be \(\sqrt{2}\), since that’s where the set stops.”

Acharya Bhaskara:
“Indeed. But here is the problem: \(\sqrt{2}\) is not in \(\mathbb{Q}\). So the rationals cannot capture the least upper bound of this set.”

3.2.1 🔹 Supremum

Acharya Bhaskara:
“This brings us to another definition:

Definition (Least Upper Bound / Supremum).
Let \(S\) be a nonempty subset of \(\mathbb{R}\). If \(S\) is bounded above then, a number \(u\) is said to be a supremum (least upper bound) of \(S\) if 1. \(s \leq u\) for all \(s \in S\), and 2. for any \(\varepsilon > 0\), there exists \(s_\varepsilon \in S\) such that \(s_\varepsilon > u - \varepsilon\).

Definition (Greatest Lower Bound / Infinimum). Let \(S\) be a nonempty subset of \(\mathbb{R}\). If \(S\) is bounded below then, a number \(u\) is said to be a infinimum (greatest lower bound) of \(S\) if 1. \(s \geq u\) for all \(s \in S\), and 2. for any \(\varepsilon > 0\), there exists \(s_\varepsilon \in S\) such that \(s_\varepsilon < u + \varepsilon\).

“So the problem with rationals is that some sets don’t have a supremum inside them?”

Acharya Bhaskara:
“Precisely. The rationals are dense but not complete. The missing piece is the completeness property of the real numbers.”

3.3 🔹 Completeness Axiom

Acharya Bhaskara:
“The defining property of the real numbers is this:

Completeness Axiom: Every nonempty set of real numbers that is bounded above has a least upper bound (supremum) in \(\mathbb{R}\).

This axiom is what makes the real line whole.”

Lila:
“So for \(S = \{ x : x^2 < 2 \}\), the supremum is \(\sqrt{2}\), which does exist in \(\mathbb{R}\).”

Acharya Bhaskara:
“Exactly. And that, Lila, is the difference between rationals and reals. Completeness fills the gaps and makes the real line continuous. Without it, calculus itself would crumble.”

3.4 ✅ Try it yourself

  1. (Optional) Give an example of a subset of \(\mathbb{Q}\) that is bounded above in \(\mathbb{Q}\) but has no least upper bound in \(\mathbb{Q}\).

3.5 Proofs

Acharya Bhaskara:
“Dear Lila, it is not enough that we make statements about numbers. We need to prove them too!”

Lila:
“But Acharya, why do we need to prove them? If something feels obviously true, can’t we just accept it?”

Acharya Bhaskara:
“Ah, that is the difference between intuition and mathematics. Intuition is helpful, but it can also mislead us. Proofs are what give mathematics its certainty. Without proof, we have only belief. With proof, we have knowledge.”

Lila:
“Can you give me an example?”

Acharya Bhaskara:
“Certainly. Take the statement: ‘The square root of 2 is irrational.’ Long ago, many believed that every number was a fraction. But through proof, we know with certainty that \(\sqrt{2}\) cannot be expressed as \(\tfrac{p}{q}\) with integers \(p\) and \(q\).”

Lila:
“I see — so proofs are like the backbone that holds mathematics upright.”

Acharya Bhaskara:
“Exactly! Proofs force us to use definitions carefully and follow logical steps. They protect us from hidden assumptions. And, more importantly, they reveal why a statement is true, not just that it is true.”

Lila:
“So every time we state something in real analysis — like that every bounded set has a supremum — we must also prove it?”

Acharya Bhaskara:
“Indeed. That completeness axiom we discussed is itself an assumption. But once accepted, we must prove all consequences from it rigorously. That is the art of analysis: building a tower of truth, brick by brick, proof by proof.”

“In order to prove some results we will first write some equivalent statements that can be used interchangeably”.
